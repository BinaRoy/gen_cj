# Engineering Governance

This directory contains the long-term operating system for project execution.

## Structure
- `global-rules.md`: hard rules shared by all engineering tasks.
- `modules/`: phase-specific playbooks.
- `templates/`: reusable response and handover templates.
- `references/reference-index.md`: categorized Cangjie references.
- `changelog/`: governance rule evolution logs.

## Rule of Use
- For non-discussion engineering work, follow `global-rules.md` and one or more module playbooks.
- If a rule changes, write a changelog entry first, then update the target playbook/template.
